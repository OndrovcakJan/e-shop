# E-shop

A modern e-commerce web application built with React, TypeScript, and Tailwind CSS. Product data is fetched from the [Fake Store API](https://fakestoreapi.com).

## Pages

- **Home** — hero banner, product listing with category filtering, search, and sorting
- **Product Detail** — product info, image lightbox, quantity selector, add to cart, related products
- **Contact** — contact info cards, form with validation, embedded map
- **Checkout** — shipping info, shipping method selection, payment details, order summary

## Features

- Shopping cart with localStorage persistence
- Category filtering and product search
- Responsive layout with hamburger menu on mobile
- Star ratings, breadcrumb navigation, image lightbox

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS v4
- React Router v7
- Axios
- Lucide React

## Getting Started

```bash
npm install
npm run dev
```
